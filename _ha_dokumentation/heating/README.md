# Heating-Dokumentation

Diese Ablage enthaelt die technische Dokumentation der Heating-App in Home Assistant.
Sie orientiert sich an der bestehenden Struktur unter `D:\Codex\_ha_dokumentation`
und beschreibt den aktuellen Stand aus:

- `packages/heating/heating_helpers.yaml`
- `packages/heating/heating_templates.yaml`
- `packages/heating/heating_scripts.yaml`
- `packages/heating/heating_automations.yaml`
- `packages/heating/heating_scenes.yaml`
- `dashboards/sieker_dashboard.yaml`
- `dashboards/sieker_hub.yaml`

## Inhalte

| Datei | Zweck |
| --- | --- |
| `HEATING_APP.md` | Hauptdokumentation mit Architektur, Sollprofil-Logik, Override-Verhalten, Dashboard und Troubleshooting |
| `heating_app_flows.drawio` | Bearbeitbare Flussdiagramme fuer Gesamtuebersicht, Profilentscheidung und Override-Lifecycle |

## Startpunkt

Fuer die fachliche und technische Uebersicht ist `HEATING_APP.md` der beste Einstieg.
Die Diagrammdatei `heating_app_flows.drawio` kann direkt in draw.io / diagrams.net geoeffnet
und spaeter an Architektur- oder Policy-Aenderungen angepasst werden.
